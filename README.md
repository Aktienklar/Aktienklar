# Aktienklar

**Aktienklar** ist eine interaktive Website zur Berechnung von Dividenden, Renditen und Zinseszinsen.  
Sie wurde für private Zwecke entwickelt, um Finanzentscheidungen anschaulich zu unterstützen.

---

## 💡 Funktionen
- Dividendenrechner  
- Zinseszinsrechner  
- Suchfunktion für Aktien  
- Interaktive Diagramme (Chart.js)  
- Modernes Design mit Tailwind CSS
