# Meistgehandelte Aktien – Quickstart (Vercel)
1) In Vercel ein neues Projekt erstellen und diesen Ordner hochladen (oder via Git importieren).
2) Environment Variables setzen:
   - FINNHUB_API_KEY = <dein Key>
   - SYMBOLS = AAPL,MSFT,NVDA,AMZN,TSLA,META,GOOGL,ORCL,INTC,AMD
3) Redeploy auslösen. API testen: /api/top-traded?period=day
4) Website öffnen und zum Abschnitt „Meistgehandelte Aktien“ scrollen.
